Module Core.Algorithm

This libary contains the implementation of core translating algorithm.

module FormulaSet : Stdlib.Set.S with type elt = Ltl.formula
module TransitionGraph : sig ... end
type state = FormulaSet.t
module StateSet : Stdlib.Set.S with type elt = state
type states = StateSet.t
val next : state -> state

next state returns {α | Xα ∈ state}

val sigma : state -> FormulaSet.t

sigma state returns the intersection of all p and ¬p in Z, p ∈ AP.

val is_reduced : state -> bool

is_reduced state returns true if the state is reduced.

val is_maximal : Ltl.formula -> state -> bool

is_maximal phi state returns true if the formula phi is maximal in state, i.e. phi is not a sub-formula of any other such formula of state.

val red : state -> states

red state returns Red(Y) = {Z reduced | Y ⟶{^*} Z}
