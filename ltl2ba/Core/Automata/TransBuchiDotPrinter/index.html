Module Automata.TransBuchiDotPrinter

Contains convenient functions to print a TransBuchi in the Dot format

It uses the using Graph.Graphviz.Dot functor to the implementation.

val fprint_graph : Stdlib.Format.formatter -> TransBuchi.t -> unit

fprint_graph ppf ba pretty prints the automata ba in the CGL language on the formatter ppf.

val output_graph : Stdlib.out_channel -> TransBuchi.t -> unit

output_graph oc ba pretty prints the automata ba in the dot language on the channel oc.
